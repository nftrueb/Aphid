# Aphid 
An LC-3 tool suite 

## Overview 
This repo contains an assembler, linker, and emulator for Little Computer 3 (LC-3). 

## Assembler 
Flow: 
- Load contents of input file 
- trim whitespace and comments (anything after ;)
- parse each line for labels, instructions, directives
- output bytecode
